Numerical Types

ACTest.I32 — Type
I32

Alias of Integer type (32 bit).

See also: R32, N32.

ACTest.I64 — Type
I64

Alias of Integer type (64 bit).

See also: R64, N64.

ACTest.API — Type
API

Alias of Integer type (Arbitrary Precision Integer).

See also: APF, APC.

ACTest.F32 — Type
F32

Alias of Float type (32 bit).

See also: R32, N32.

ACTest.F64 — Type
F64

Alias of Float type (64 bit).

See also: R64, N64.

ACTest.APF — Type
APF

Alias of Float type (Arbitrary Precision Float).

See also: API, APC.

ACTest.C32 — Type
C32

Alias of Complex type (32 bit).

See also: R32, N32.

ACTest.C64 — Type
C64

Alias of Complex type (64 bit).

See also: R64, N64.

ACTest.APC — Type
APC

Alias of Complex type (Arbitrary Precision Complex).

See also: API, APF.

ACTest.R32 — Type
R32

Alias of Integer and Float types (32 bit). Here R means Real.

See also: N32, N64, APN.

ACTest.R64 — Type
R64

Alias of Integer and Float types (64 bit). Here R means Real.

See also: N32, N64, APN.

ACTest.APR — Type
APR

Alias of Integer and Float types (Arbitrary Precision). Here R means Real.

See also: N32, N64, APN.

ACTest.N32 — Type
N32

Alias of Integer, Float, and Complex types (32 bit). Here N means Number.

See also: R32, R64, APR.

ACTest.N64 — Type
N64

Alias of Integer, Float, and Complex types (64 bit). Here N means Number.

See also: R32, R64, APR.

ACTest.APN — Type
APN

Alias of Integer, Float, and Complex types (Arbitrary Precision). Here N means Number.

See also: R32, R64, APR.
