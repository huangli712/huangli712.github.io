Grids

Grids on imaginary axis.

In the ACFlow package, the input correlators are defined on grid, while the calculated spectral functions are defined on mesh. The ACFlow package supports both imaginary-time and Matsubara frequency grids.

AbstractGrid

An abstract type representing the imaginary axis. It is used to build the internal type system.

ACFlow.FermionicImaginaryTimeGrid — Type
FermionicImaginaryTimeGrid

Mutable struct. It represents the fermionic imaginary time grid.

Members

  • ntime -> Number of time slices.
  • β -> Inverse temperature.
  • τ -> Vector of grid points.

See also: FermionicMatsubaraGrid.
